Welcome

This is the WalledCTTY presence on the modern network. We've styled our website to make it feel a little more familiar to those coming from the old net. For those visitors, be careful out here. There's danger lurking within the code. Corporations, insidious and omnipresent, lie in wait for any vulnerability to siphon your data and, indeed, your very essence. For anyone wishing to escape the corporate web, port 1989 at bbs.walledctty.com is accepting telnet connections. More information about this can be found on the connect page.

If you want information about what WalledCTTY is, as well as the software and hardware used to make it work, those details can be found on the about page. A feed to the non-networked message areas has been developed, the output of which is available on the messages page. Anchor points are provided should anyone feel the need to link directly to a specific message. This is a one-way connection. To post content or reply to a message, you'll need to log into WalledCTTY itself. This is a feature, not a bug. Also, a list of current games can be found on the doors page.

Caveats
Unlike many modern telnet boards that run on relatively up-to-date hardware and software (or on older software running in a speedy virtual machine), WalledCTTY is vintage software running on vintage hardware. Most of this stuff is thirty-years-old or older. Some tasks may require a bit more patience due to the system's older technology. If anything, this makes the whole thing feel a bit more legit.
Currently the system has *only* been tested with 80 columns using modern telnet clients. 40-column, non-ASCII/ANSI testing will happen sooner, rather than later.
