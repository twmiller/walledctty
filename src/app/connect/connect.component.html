"Modern" Telnet Connections

To connect to WalledCTTY BBS using a modern computer, you'll need to install a telnet client for your given computer and telnet to: bbs.walledctty.com on port 1989. I highly recommend using something with ANSI support for the best experience. I believe SyncTerm is available for Windows, Linux, OpenBSD, NetBSD, OS X, and FreeBSD and works really well. On Windows and Linux machines, Netrunner is another option. On macOS, there's also MuffinTerm, which I've used for testing the board, and it also works very well. If you want a retro feel, you even have the option to emulate CRTs in MuffinTerm. :D

Node 1 Modem Connection

For users inside of the US, one node is connected to an actual modem. The modem is an external U.S. Robotics 56k v.92 answering calls at 1 (970) 449-0767. This is completely untested though. I have tried the modem itself on *other* dial-up boards that still exist, though, and was able to connect with success. I've found that faster modems (with error correction) actually tend to work better on VOIP than slower modems without. If you try to dial-in and it doesn't work, log-in via telnet and let me know!
